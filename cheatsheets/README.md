Concise, high-signal reminders for shortcuts, flags, and workflows that you need to recall fast.

| File | Focus |

| `systemd-cheatsheet.md` | Service management, unit queries, journald snippets. |

Add additional sheets when a topic grows beyond a couple of bullets inside other docs.

## Writing Guidelines

- Keep entries tightly scoped; bullets or two-column tables work best.
- Prefer action-oriented phrasing (`Reload all units: ...`) over prose.
- Use fenced `bash` blocks sparingly for multi-line snippets; otherwise inline code is sufficient.
- Cross-link to the deeper command or troubleshooting pages when a cheat item needs more explanation.
